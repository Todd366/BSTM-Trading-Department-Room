Methodology

The BSTM 8-Module Analysis Framework

A top-down, multi-confluent analysis system applied before every trade. Not prediction — process.

MODULE 01
🔢
Psychological Levels
Major round numbers and historical S/R levels on D1/H1. How price has reacted previously at each level.
MODULE 02
📐
Market Structure
D1 / H1 / M15 bias. BOS, CHOCH, CISD identification. Is the dominant structure intact or broken?
MODULE 03
🌊
Order Flow
Which side shows more aggression? Imbalance or exhaustion candles. Absorption on M15 and M1.
MODULE 04
📈
Trend Line Liquidity
Key trend lines with touch count. Liquidity sweeps (equal highs/lows, stop hunts). Post-sweep reversals.
MODULE 05
⬜
Fair Value Gaps
All visible unfilled FVGs. Price position relative to FVG boundary. Support or resistance classification.
MODULE 06
🕐
True Open & True Close
Price relative to today's True Open. Prior TO/TC behaviour. Strong displacement or rejection signals.
MODULE 07
🔄
Quarterly Theory
Position inside the quarterly range. High-probability turning zones. 22-minute cycle observations.
MODULE 08
⚖️
Risk & Trade Management
Entry zone. SL ≤80 pips. Min 2 TP levels. Max $1.50 risk per position. Max 2 positions. R:R documented.
Top-Down Logic

D1 — Where is the market going?  →  H1 — How is structure behaving?  →  M15 — Where is price reacting?  →  M1 — Entry confirmation only.

Daily Rules

Completely sober. Strictly follow 3×3 Rule (max 3 setups, max 3 checks). Enter the focused state before every analysis. Risk = Survival. Execution = Final action.
